## Step 1: Search for Conflicts (Free)

### USPTO TESS Search

1. Go to: https://tmsearch.uspto.gov
2. Select "Basic Word Mark Search"
3. Search for "HERMES"
4. Filter by Class 9 (software)
5. Look for conflicts in clinical/healthcare/AI software

### What Disqualifies

| Existing Mark | Conflict? |
|---------------|-----------|
| HERMES (fashion, Hermès) | No - different class (Class 18, 25) |
| HERMES (unrelated software) | Maybe - check if similar goods |
| HERMES (healthcare software) | Yes - likely conflict |

### Our Situation

"Hermes" is used by:
- Hermès (fashion) - Class 18, 25 - **No conflict**
- Various tech projects - Check if trademark registered
- Greek mythology - Can't trademark the word generally

Our use: "HERMES" for clinical AI supervision protocols
- Specific niche
- Likely sufficiently distinct

**Proceed with filing unless you find identical mark in Class 9 for healthcare/clinical software.**

---

## Step 2: File Application

### Go to USPTO

1. URL: https://www.uspto.gov/trademarks/apply
2. Create account if needed
3. Select "TEAS Plus" ($250) - cheaper, stricter requirements

### TEAS Plus vs TEAS Standard

| Option | Cost | Requirements |
|--------|------|--------------|
| **TEAS Plus** | $250 | Must use predefined descriptions |
| TEAS Standard | $350 | Can write custom descriptions |

**Use TEAS Plus** - the predefined descriptions work for software.

### Application Fields

#### For HERMES

```
Mark: HERMES
Mark Type: Standard Characters (word mark, no logo)

Class: 009
Goods/Services Description (select from list):
  "Downloadable computer software for [integrating clinical
   data for use by healthcare providers]"

  OR closest match like:
  "Downloadable computer software for healthcare data management"

Filing Basis:
  [ ] 1(a) Use in Commerce - if already using
  [X] 1(b) Intent to Use - if not yet using publicly

Owner:
  Entity: Corporation (or LLC)
  Name: Regain Health, Inc.
  State: [Your state]
  Address: [Your address]

Correspondence:
  Email: [your email]
  Phone: [your phone]
```

#### For POPPER

```
Mark: POPPER
Mark Type: Standard Characters

Class: 009
Goods/Services Description:
  "Downloadable computer software for [evaluating safety of
   artificial intelligence applications in healthcare]"

Filing Basis: 1(b) Intent to Use

Owner: [Same as HERMES]
```

## Step 4: After Filing

### What Happens Next

| Timeline | Event |
|----------|-------|
| Day 0 | Application submitted, receive serial number |
| Month 3-6 | USPTO examiner reviews |
| Month 6-8 | Office Action (if issues) OR approval |
| Month 8-10 | Published for opposition (30 days) |
| Month 10-12 | Registration (if no opposition) |

### If You Get an Office Action

Common issues:
1. **Description too broad** → Narrow the description
2. **Likelihood of confusion** → Argue distinctiveness or abandon
3. **Missing information** → Provide requested info

**You can respond yourself or hire a lawyer at this point.**

### If Using Intent to Use (1b)

After approval, you must:
1. Actually use the mark in commerce
2. File "Statement of Use" with evidence
3. Pay $100 fee

**Evidence:** Screenshot of website, npm package, GitHub repo showing the mark.

## Common Mistakes to Avoid

| Mistake | Consequence |
|---------|-------------|
| Filing after public use | Can't use Intent to Use basis |
| Wrong classification | Rejection or wrong protection |
| Description too broad | Office Action, delays |
| Typos in mark | Have to refile |
| Wrong entity name | Assignment issues later |

## Using the Marks Before Registration

### What You CAN Do

- Use "Hermes™" and "Popper™" (TM symbol)
- Claim common law trademark rights
- Reference "trademark pending" status

### What You CANNOT Do

- Use ® symbol (only after registration)
- Claim federal registration benefits

### In Documentation

```markdown
Hermes™ is a trademark of Regain Health, Inc.
Popper™ is a trademark of Regain Health, Inc.
```
